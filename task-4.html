Використання всіх базових селекторів CSS та псевдо-класи

Сторінка, на якій використані всі базові селектори CSS та псевдо-класів. Також тут використано поєднання різних селекторів, які дають можливість більш точно задавати стилі для різних елементів.

Використав селектор a:hover для того, щоб змінити стиль посилання при наведенні курсору на нього.

Використав селектор input:focus, який дозволяє змінити стиль поля вводу при отриманні на нього фокусу.

У списку використано селектор li:nth-child(odd) та li:nth-child(even), які дозволяють змінювати стиль для парних та непарних елементів списку відповідно.

А селектор ul > li:first-child дозволяє змінювати стиль тільки першого елемента списку.

Використав поєднання різних селекторів для того, щоб задати стиль для певних елементів списку. Наприклад, селектор ul li:first-child + li задає стиль для другого елементу списку.
